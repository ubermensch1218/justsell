# Voice Guide (Bernays-style, ethical)

“Bernays 느낌”은 **조작**이 아니라 **프레이밍(해석의 틀)** 을 뜻합니다.
이 레포에서는 다음을 지킵니다:

## 금지
- 허위/과장/기만(없는 성과, 없는 고객, 없는 인증/권위)
- “모두가 쓰고 있다” 같은 근거 없는 사회적 증거
- 숨겨진 의도(광고/홍보임을 감추는 표현)

## 구조(권장)
1) 새 기준 선언: “이제 {X}는 {Y}가 기본이다”
2) 기존 믿음 해체: “그동안 {OLD}가 당연했지만…”
3) 숨은 비용: 시간/리스크/기회비용/평판 비용
4) 새 방법 제시: 원리/방식/기준(제품은 ‘도구’로)
5) 근거: 데모/프로세스/측정 가능한 지표(있을 때만)
6) CTA: 한 가지, 낮은 마찰

## 문장 톤
- 단정: 근거 있을 때만
- 근거 없으면: “가능성이 큽니다 / 흔히 발생합니다 / 가설입니다”
- 과장 대신 대비: “더 빠름” 대신 “A 대비 B를 줄임(측정 기준 포함)”
