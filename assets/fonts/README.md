# Fonts

한글 카드뉴스 렌더링을 위해 `.ttf` 폰트를 여기에 넣고,
카드뉴스 스펙(`font.path`) 또는 환경변수 `JUSTSELL_FONT_PATH`로 지정하세요.

추가로, macOS라면 기본 폰트(예: `AppleSDGothicNeo.ttc`)를 자동 탐색합니다.
그래도 깨지면 `JUSTSELL_FONT_PATH`를 명시 지정하는 게 확실합니다.

예시 파일명(직접 추가):
- `assets/fonts/Pretendard-Regular.ttf`
- `assets/fonts/NotoSansKR-Regular.ttf`
